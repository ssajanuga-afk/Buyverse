Button Component System

Solid Color Buttons

Standard buttons with solid background colors

<button class="btn btn-primary">Primary</button> <button class="btn btn-success">Success</button> <button class="btn btn-danger">Danger</button>

Outline Buttons

Buttons with transparent backgrounds and colored borders

<button class="btn btn-outline-primary">Primary</button> <button class="btn btn-outline-success">Success</button>

Button Sizes

Different button dimensions

<button class="btn btn-success btn-sm">Small</button> <button class="btn btn-success">Normal</button> <button class="btn btn-success btn-lg">Large</button>

Button States

Different button states

<button class="btn btn-success">Normal</button> <button class="btn btn-success active">Active</button> <button class="btn btn-success" disabled>Disabled</button>

Buttons with Icons

Combine icons with buttons

<button class="btn btn-success"> <i class="fa fa-download"></i> Download </button> <button class="btn btn-dark btn-icon"> <i class="fa fa-heart"></i> </button>

Gradient Buttons

Stylish gradient background buttons

<button class="btn btn-gradient">Gradient Green</button>

Special Buttons

Unique and special button styles

<button class="btn btn-success btn-pill">Pill Button</button> <button class="btn btn-primary btn-raised">Raised Button</button> <button class="btn btn-secondary btn-flat">Flat Button</button>

Buttons with Badges

Add badges or notifications to buttons

<button class="btn btn-success btn-with-badge"> <i class="fa fa-bell"></i> Notifications <span class="btn-badge">5</span> </button>

Button Groups

Group buttons together

<div class="btn-group"> <button class="btn btn-success">Left</button> <button class="btn btn-success">Center</button> <button class="btn btn-success">Right</button> </div>

Full Width Buttons

Block level buttons

<button class="btn btn-success btn-lg btn-block"> <i class="fa fa-sign-in"></i> Login </button>

Dynamic button interactions using JavaScript

Loading State
Toggle Button
Count Badge
Disable/Enable
ButtonUtils.showLoading('asyncBtn'); // ... perform operation ... ButtonUtils.hideLoading('asyncBtn'); ButtonUtils.disable('toggleTargetBtn'); ButtonUtils.enable('toggleTargetBtn'); ButtonUtils.updateBadge('badgeBtn', newCount);
