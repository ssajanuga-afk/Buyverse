3️⃣ JavaScript: Button Functionality

Include JavaScript File:

<script src="assets/js/buttons.js"></script>

Basic Click Handler:

ButtonUtils.on('myBtn', function() { alert('Button clicked!'); });

Show Loading State:

<button id="submitBtn" class="btn btn-success"> Submit </button> <script> ButtonUtils.on('submitBtn', function() { ButtonUtils.showLoading('submitBtn', 'Processing...'); // Simulate work setTimeout(function() { ButtonUtils.hideLoading('submitBtn'); alert('Done!'); }, 2000); }); </script>
Live Demo:

Add Badge to Button:

<!-- Create button with ID --> <button id="cartBtn" class="btn btn-warning"> <i class="fa fa-shopping-cart"></i> Cart </button> <script> // Add badge with count ButtonUtils.addBadge('cartBtn', 5); // Or update existing badge ButtonUtils.updateBadge('cartBtn', 10); // Remove badge ButtonUtils.removeBadge('cartBtn'); </script>

Disable/Enable Button:

ButtonUtils.disable('myBtn'); // Disable button ButtonUtils.enable('myBtn'); // Enable button // Check state if (ButtonUtils.isDisabled('myBtn')) { console.log('Button is disabled'); }

Create Button Dynamically:

let btn = ButtonUtils.create({ className: 'btn btn-success btn-lg', text: 'Dynamic Button', id: 'dynBtn', onClick: function() { alert('I was created with JavaScript!'); } }); document.body.appendChild(btn);

All ButtonUtils Methods:

Loading:
showLoading(id) • hideLoading(id)
State:
disable(id) • enable(id) • isDisabled(id)
Text:
setText(id, text)
Badge:
addBadge(id, count) • updateBadge(id, count) • removeBadge(id)
Events:
on(id, callback) • async(id, asyncFunc)
Creation:
create(options) • createGroup(buttons)
