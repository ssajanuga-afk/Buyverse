Button Component System

Quick Start

1. Include Files in Your HTML

<!-- CSS --> <link rel="stylesheet" href="assets/css/buttons.css"> <!-- JavaScript --> <script src="assets/js/buttons.js"></script>

2. Use a Button

<button class="btn btn-success">Click Me</button>

3. Interact with JavaScript

ButtonUtils.on('myBtn', function() { alert('Button clicked!'); });
✅ That's it! Your buttons are now fully functional with all features.

JavaScript API Reference

Core Methods

Method Description Example
showLoading(id, text) Show loading state ButtonUtils.showLoading('btn', 'Loading...')
hideLoading(id) Hide loading state ButtonUtils.hideLoading('btn')
disable(id) Disable button ButtonUtils.disable('btn')
enable(id) Enable button ButtonUtils.enable('btn')
setText(id, text) Change button text ButtonUtils.setText('btn', 'New Text')
on(id, callback) Add click handler ButtonUtils.on('btn', function() { })
addBadge(id, count) Add badge to button ButtonUtils.addBadge('btn', 5)
updateBadge(id, count) Update badge count ButtonUtils.updateBadge('btn', 10)

Common Use Cases

Example 1: Loading Button

ButtonUtils.showLoading('submitBtn', 'Processing...'); // Do some work... setTimeout(() => { ButtonUtils.hideLoading('submitBtn'); }, 2000);

Example 2: Notification Badge

let count = 0; function addNotification() { if (count === 0) { ButtonUtils.addBadge('notifyBtn', 1); } else { ButtonUtils.updateBadge('notifyBtn', count + 1); } count++; }

Example 3: Async Operation

ButtonUtils.async('submitBtn', async function() { let response = await fetch('/api/submit'); let data = await response.json(); console.log(data); });

Example 4: Create Button Dynamically

let btn = ButtonUtils.create({ className: 'btn btn-success', text: 'Dynamic Button', id: 'dynBtn', onClick: function() { alert('Clicked!'); } }); document.body.appendChild(btn);

Tips & Tricks

💡 Pro Tip 1: Always use .btn class with a color class like .btn-success
💡 Pro Tip 2: Combine classes for powerful styling: <button class="btn btn-success btn-lg btn-pill">
💡 Pro Tip 3: Use button states with JavaScript: ButtonUtils.disable(), ButtonUtils.showLoading()
💡 Pro Tip 4: Enable ripple effect globally: ButtonUtils.enableRippleForAll();
💡 Pro Tip 5: Always add IDs to buttons for JavaScript interaction
